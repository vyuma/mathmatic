Available Shortcuts

The MathField component includes these keyboard shortcuts:

  • alpha → α (\\alpha)
  • beta → β (\\beta)
  • gamma → γ (\\gamma)
  • sum → ∑ (\\sum)
  • int → ∫ (\\int)
  • frac → fraction template (\\frac{}{})
  • sqrt → square root template (\\sqrt{})
  • lim → limit template (\\lim_{})
  • infty → ∞ (\\infty)
